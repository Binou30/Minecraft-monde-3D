## Minecraft Monde 3D :

J'ai réalisé ce projet dans le but de découvrir la 3D avec Python et Ursina, et franchement, le rendu est vraiment top! En suivant le mode d'emploi, vous trouverez un monde réalisé avec des cubes aux textures de Minecraft dans lequel vous pourrez vous balader en caméra vue du dessus. Il y a même des mobs!

Bonne exploration!

## Mode d'emploi :

Dans ce dossier, vous trouverez minecraft.py, un fichier python qui vous permettra, si vous avez installé le module Ursina (en tapant dans le terminal selon, votre version de pip : pip --install ursina ou pip3 --install ursina) de vous balader en caméra dans un petit monde Minecraft en 3D (il est figé, c'est juste pour faire joli). Cependant, avec les nouvelles version de Python, le chargement des textures ne se fait pas très bien donc mettez dans les issues si vous trouvez des bugs!
