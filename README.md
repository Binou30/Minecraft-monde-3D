## Minecraft Monde 3D :

Dans ce dossier, vous trouverez minecraft.py, un fichier python qui vous permettra, si vous avez installé le module Ursina (en tapant dans le terminal selon, votre version de pip : pip --install ursina ou pip3 --install ursina) de vous balader en caméra dans un petit monde Minecraft en 3D (il est figé, c'est juste pour faire joli).

Bonne exploration!
